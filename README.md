# clickdev.uz

Сайт-визитка Click Dev. Чистая статика: `index.html` (разметка+стили+i18n), `assets/` (SVG из брендстайл-канона Click Dev).

- Тексты: словарь `I18N` в `index.html` (ru/uz/en). ⚠️ UZ — черновой перевод, нужна вычитка носителем.
- Деплой: push в main → GitHub Pages. Домен: clickdev.uz (CNAME).
- Брендстайл: канон в clickdev-company/operations/Бренд.
